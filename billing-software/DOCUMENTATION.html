Maple Super Market

Billing Software - Technical Documentation

1. Project Overview

Maple Super Market Billing Software (known as Maple Pro) is an advanced, responsive retail management system. Designed for high-speed counter operations and comprehensive backend management, it streamlines the entire lifecycle of a retail business—from stock procurement to customer checkout and daily financial closure.

Software Focus: Performance, Rich Visuals, and Detailed Reporting.

2. Technology Stack

The application is built using a modern, scalable stack:

  • Backend Python with Flask (Migrated from FastAPI for enhanced flexibility)
  • Database MySQL (Structured relational data storage)
  • Frontend Semantic HTML5, Vanilla CSS3, ES6+ JavaScript
  • Printing Direct Thermal Printing (ESC/POS compatible via browser)
  • Animations Custom CSS3 Keyframes & JS Animation Hooks

3. Key Features & Modules

3.1 Advanced Point of Sale (POS)

The billing interface is the heart of the system. It features:

  • Fast barcode/QR scanning integration.
  • Automatic calculation of Bizz Points and TSC based on category rules.
  • Keyboard-first workflow for rapid billing.
  • Multi-mode payment support (Cash, UPI, Card, Credit).

3.2 Inventory & Stock Control

Complete control over products and logistics:

  • Product Master management with unit-wise pricing.
  • Stock Transfer module for Godown-to-Shop movements.
  • Automatic stock decrement on sales and increment on returns.

3.3 Returns & Voids

Secure transaction management:

  • Rename of "Returns" to "Returns & Voids" to handle full bill cancellations.
  • Reprocessing logic: Items can be returned and the remaining bill automatically re-processed.

4. Comprehensive Reporting System

The software provides depth with 22 distinct report templates:
